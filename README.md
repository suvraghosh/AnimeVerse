# Live Demo
### [Click Here](https://anime-versex.vercel.app/)


## Run Locally

Clone the project

```bash
  git clone https://github.com/suvraghosh/AnimeVerse.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
